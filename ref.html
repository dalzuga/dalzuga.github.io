  • How to delete all tables and users

    To delete all users:
    mysql -uroot -p -e "select User,Host from mysql.user;" | grep -v User | grep -v root | grep -v debian-sys-maint | gawk '{print "drop user " $1"@"$2";select sleep(0.1);"}'

    To delete all tables:
    mysql -uroot -p -e "show databases" | grep -v Database | grep -v mysql | grep -v information_schema | gawk '{print "drop database " $1";select sleep(0.1);"}'

    Don't forget to FLUSH PRIVILEGES;

    Reference:

    • Drop All Databases in MySQL | Mohinish's Blog
